# AI Kiosk - 도서관 책 추천 큐레이터

동양미래대학교 도서관을 위한 AI 기반 스마트 책 추천 키오스크 시스템

## 프로젝트 소개

사용자의 관심사와 감정 상태를 분석하여 최적의 도서를 추천하는 인터랙티브 키오스크 애플리케이션입니다.
알라딘 도서 API와 OpenAI GPT를 결합하여 개인화된 큐레이션 서비스를 제공합니다.

## 주요 기능

### 맞춤 추천
- 관심 키워드 기반 도서 검색
- 독서 목적별 추천 (학습, 취미, 자기계발, 휴식)
- AI 큐레이터의 상세한 추천 이유 제공

### 기분별 추천
6가지 감정 상태에 맞춘 도서 추천
- 힐링 / 설렘 / 우울 / 호기심 / 지침 / 성장

### 도서 검색
- 키워드, 제목, 저자, 출판사 검색
- 15개 카테고리별 필터링
- 베스트셀러 & 신간 도서 조회

## 기술 스택

### Backend
- **Flask 3.0.0** - 웹 서버 및 API
- **OpenAI GPT-4o-mini** - AI 기반 추천 엔진
- **알라딘 API** - 도서 정보 제공

### Frontend
- **Vanilla JavaScript** - 클라이언트 로직
- **HTML5 / CSS3** - 프리미엄 키오스크 UI
- **Pretendard 폰트** - 한글 최적화 타이포그래피
- **Font Awesome** - 아이콘 시스템

## 프로젝트 구조

## API 엔드포인트

| 엔드포인트 | 메서드 | 설명 |
|-----------|--------|------|
| `/` | GET | 메인 키오스크 페이지 |
| `/api/search` | GET | 도서 검색 |
| `/api/bestsellers` | GET | 베스트셀러 조회 |
| `/api/new-releases` | GET | 신간 도서 조회 |
| `/api/recommend` | POST | AI 맞춤 추천 |
| `/api/recommend/mood` | POST | 기분별 추천 |

## UI/UX 특징

- **다크 모드 디자인** - 프리미엄 키오스크 경험
- **터치 친화적** - 큰 버튼과 명확한 인터랙션
- **반응형 레이아웃** - 다양한 화면 크기 지원
- **로딩 애니메이션** - 부드러운 사용자 경험
- **키오스크 모드** - 우클릭, 드래그, 텍스트 선택 방지
